Creating the Bot on Discord
1. Go to https://discord.com/developers/applications
2. Click New Application.
3. Name the Application and create it.
4. Click "Bot" Tab on the left side of the page.
5. Click Add Bot and copy token by pressing copy under the "Token" heading.
6. This is your BOT_TOKEN keep it safe.

Invite Bot to server
1. Click OAuth2 Tab from the left of the page of your created Application.
2. Under Scope section check the box that says bot.
3. Copy the url generated and go to the URL.
4. Select the server to add the bot into and press Authorize.
